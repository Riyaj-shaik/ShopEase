🛍️ ShopEase – E-Commerce Web Application
ShopEase is a full-stack e-commerce web application designed to provide a seamless online shopping experience. It integrates product browsing, cart management, secure checkout, and order tracking into one cohesive platform.

📌 Features
User Management – Registration, login

Admin Dashboard – Manage products, add, update, delete products. 

Product Catalog – Browse products with categories, filters, and search.

Shopping Cart – Add and remove items.

Responsive Design – Works on desktops, tablets, and mobile devices.

🛠️ Technology Stack

Frontend

HTML5, CSS3, JavaScript

Bootstrap / Tailwind CSS

Backend

Spring Boot (Java)

RESTful APIs

Hibernate / JPA

Database

MySQL

Other Tools
Maven

Eclipse IDE

Git & GitHub

Copy

cd ShopEase


Backend Setup

Open Backend/ProjectEcommerce in Eclipse or IntelliJ IDEA.

Configure application.properties with your MySQL credentials.

Run the Spring Boot application.

Frontend Setup

Run Login.html

Access the Application

Frontend: http://localhost:3000
